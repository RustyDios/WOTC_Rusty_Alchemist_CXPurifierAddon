Copy of Steam Desc, in case it gets overriden
=============
This mod adds a new chemthrower, with three tiers, that have a chance to drop from the Purifiers and Purifier Captains. These weapons will drop as 'corpse loot' and as thus can only be collected on missions that corpses can be collected on.
The weapons have the dark black looks of the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1958853933] CX Purifier Flamers [/url] but can be tinted. 

Also adds two new unique chemthrowers that can be dropped from a [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1731235406] CX Bio Division [/url] Bio Mec Trooper. These unique weapons have a green look and can shoot either acid or poison.

They can be used by anyone that can use Chemthrowers, and thus my [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1922770319] Alchemist Class [/url] is not a Hard Requirement as long as you have [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1918448514] Mitzruti's Base Immolator/Chemthrower Mod[/url].

Adds a proving ground project after Heavy Plasma/Beam Cannon weapons have been researched that has a chance to give one of these new chemthrowers also, as a backup incase you haven't found any in drops OR don't wish to use the expanded enemies.
The project gives superior tier chemthrowers and includes the Cryolator if you have [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1891339003] Bitterfrost Protocol [/url].

Adds an [i]OPTIONAL[/i] proving ground project to make all canisters come from an experimental project. Defaults to not active.

[b] I highly advise that the required mods be subbed to first. [/b]

[h1]===== Config =====[/h1]
Most weapon and ability values can be adjusted in the XComXenolator.ini config file
The drop source can be configured to drop from the standard Purifier or Purifier Captains, defaults to both.
The drop rates can be configured in the XComGameCore.ini file

Tintable weapon models are enabled by default for Xenolators. New models are enabled for Immolators and Cryolators.
Can be toggled back to old models in the config (including toggling Immolators and Cryolators to tintable models).
[i]The tintable weapons will start out with the default colour set in your ini's (likely highlander config)[/i]

[h1]===== Known Issues =====[/h1][list]
[*]  [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1171964288] Configure Upgrade Slots [/url]. [strike] This mod will mean that you will not be able to attach 'normal' weapon upgrades to chemthrowers, they can use the special made attachments instead. [/strike] Allowed attachments for chemthrowers have now changed, making this issue obsolete.
[*] Sometimes visualization of some chemthrower abilities glitches out, they still work functionally but don't display a stream-jet.

[*] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2237137501] Iridars Immolator Overhaul [/url] Also changes the archytype of Immolators. To get these two mods to work together, you need to make adjustments to this mods config files.
XComWeaponSkinReplacer.ini needs my section for Immolators removed and XComXenolator.ini needs to be set to NOT update Immolator models.

[*] Special chemthrower attacks (like Medispray, Fumigate and Fulmination) all take their cone sizes from the 'Immolator', the default ranges of these chemthrowers might not match

[*] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1142377029] Grimy's Loot Mod [/url][i]conflicts with [/i] [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2042829719] UI Tool[url]

You [b]CAN[/b] use this mod without the Weapon Upgrade UI Tool if you wish to use Grimy's Loot Mod, it simply means that your chemthrowers won't align correctly in the Weapon Upgrade Menu....

[*] Bug reports welcome[/list]
